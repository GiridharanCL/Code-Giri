SERVO MOTOR INTERFACE:

Construct a model asynchronously from a compiled model asset.

Installing Servo Library To install the library, navigate to Sketch > Include Library > Manage Libraries, wait for the Library Manager to download the libraries index and update the list of installed libraries. Filter your search by entering ‘Servo’. Look for desired library for keypad. Click on that entry and then choose Install. Then need to enable the header file (#include< Servo.h>), this header file has instructions written in it, which enables the user to interface. Here make the SERVO Pin 13 HIGH. The following code will turn a servo motor to 0 degrees, wait 1 second, then turn it to 90, wait one more second, turn it to 180, and then go back. The operation of Servo Motor Interface is explained step by step in C code given below .


Parameters


  

    
    
  
    //……………….KRISH_TEC……………………
    //………………. SERVO MOTOR INTERFACE …………………….
    
    #include <Servo.h> // Include the Servo library
    int servoPin = 13; // it declares a variable named servoPin as an integer using digital pin 13.
    Servo Servo1; // create a servo object
    void setup() {
    Servo1.attach(servoPin);  // attach the servo to the used pin number
    }
    void loop(){
      Servo1.write(0);  // make servo go to 0 degrees
    delay(2000);  // wait for a second
    Servo1.write(90);  // make servo go to 90 degrees
    delay(2000);  // wait for a second
    Servo1.write(180);  // make servo go to 180 degrees
    delay(2000);  // wait for a second
    }

Uploading the Code from the Computer to the Arduino Board:

Connect the board to computer via USB cable. Next, go to Tools->Port and select the correct port number. In windows, it starts with COM which is followed by a number. Then select the board by going to Tools->Board. Here select the Arduino Uno board. Finally, click on the Upload button (it’s the button with the right-facing arrow). After Uploading the code, the Servo1.write(0) function will make the servomotor spin counter-clockwise at full speed. The Servo1.write(90) function will stop the motor and Servo1.write(180) will turn the motor clockwise at full speed.
