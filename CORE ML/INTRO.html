Introduction :


The KT-ARDK is an ALL-IN-ONE DEVELOPMENT KIT. It is the perfect kit to get familiar with electronics as well as coding. This versatile microcontroller is equipped with the ATmega328 Controller. This kit will give you a great experience.


KT-ARDK Kit contains,:


1. Power Supply
2. ATMEGA UC session & shield interface
3. USB programmer
4. UART
5. I2C RTC & EXT RTC
6. SPI – Serial Peripheral Interface
7. Analog interface
8. 7 Segment display with shift register
9. Expansion ports
10. RGB LED interface
11. LCD interfacing
12. 4*4 Keyboard interface
13. Relay, Buzzer and Servo motor interface
14. Motor interfaces (DC Motor & Stepper Motor)
15. IR remote interface

Power Supply :


This block consists of L7805CV, switch, external port, LED, resistors and capacitors. L7805CV is a three terminal linear voltage regulator IC with a fixed output voltage of 5V which is useful in a wide range of applications. If the entire PCB board gets power on, that LED starts glowing.


ATMEGA UC session & Shield Interface :


This block consists ATmega328 microcontroller, quartz crystal oscillators, power pins, analog pins, digital (PWM) pins. In KT-ARDK-040515 kit we use ATmega328 microcontroller. This microcontroller is a single chip created by Atmel in the megaAVR family (AVR stands for Alf and Vegard's RISC processor, AVR generally refers to the 8-bit RISC line of Atmel AVR microcontrollers). It has a modified Harvard architecture 8-bit RISC processor core. Due to RISC architecture, an AVR microcontroller can execute 1 million instructions per second if the cycle frequency is 1 MHz provided by a crystal oscillator.

Pin Functions - ATmega328 Microcontroller :


★ 20 of the pins function as I/O ports. This means they can function as an input to the circuit or as output. Whether they are input or output is set in the software.
★ 14 of the pins are digital pins, of which 6 can function to give PWM output. 6 of the pins are for analog input/output.
★ 2 of the pins are for the crystal oscillator. This is to provide a clock pulse for the ATmega chip.
★ A clock pulse is needed for synchronization so that communication can occur in synchrony between the ATmega chip and a device. The chip needs power so 2 of the pins, Vcc and GND, provide it power so that it can operate. The ATmega328 is a low-power chip, so it only needs between 1.8-5.5V of power to operate.
★ The ATmega328 chip has an analog-to-digital converter (ADC) inside of it. This must be or else the ATmega328 wouldn't be capable of interpreting analog signals. Because there is an ADC, the chip can interpret analog input, which is why the chip has 6 pins for analog input.
★ The ADC has 3 pins set aside for it to function, AVCC, AREF, and GND. AVCC is the power supply, positive voltage, that for the ADC. The ADC needs its own power supply in order to work. GND is the power supply ground. AREF is the reference voltage that the ADC uses to convert an analog signal to its corresponding digital value.
★ Analog voltages higher than the reference voltage will be assigned to a digital value of 1, while analog voltages below the reference voltage will be assigned the digital value of 0.
★ Since the ADC for the ATmega328 is a 10-bit ADC, meaning it produces a 10-bit digital value, it converts an analog signal to its digital value, with the AREF value being a reference for which digital values are high or low.
★ The last pin is the RESET pin. This allows a program to be rerun and start over. And this sums up the pinout of an ATmega328 chip.

USB Programmer :


This block consists of FTDI IC, external USB port, LEDs, resistors and capacitors. FTDI chip’s integrated circuit development is guided by a simple principle, making design easy. This translates into taking into account not just the chip design, but also system integration and software needs, value-added features and design support. If data transfer happens those LEDs start glowing.



UART :


This block consists of LEDs, resistors and 4 input pins and 2 output pins. Those 4 input pins are VCC, ground, TX(transmitter) and RX (receiver). 2 output pins are TX (transmitter) and RX (receiver). If data transfer happens those LEDs start glowing.



I2C RTC & EXT RTC :


This block consists of an IN1307N IC, battery, capacitor, 2 resistors, 4 input pins and 2 output pins. Those 4 input pins are VCC, ground, SCL (Serial Clock Pin) and SDA (Serial Data Pin). 2 output pins are SCL (Serial Clock Pin) and SDA (Serial Data Pin). IN1307N is a CMOS IC of real time watch with serial interface.



SPI – Serial Peripheral Interface :


The Serial Peripheral Interface (SPI) is a synchronous serial communication interface specification used for short-distance communication. This block consists of 4 input pins. Those 4 input pins are SS (Slave Select), MISO (Master In Slave Out), MOSI (Master Out Slave In), SCK (Slave Clock).

Analog Interface :


An analog interface is point-to-point. This block consists of LM35 (Temperature Sensor), potentiometer (POT), external ADC pins. Potentiometer used for adjustment, calibration and tuning purpose in circuits. This block has switch ports to connect anyone of the above mentioned analog interface.

7 Segment Display with shift register :


This block consists of 4 BC547B transistors acting as switch between all four 7 segment display quick switching and pulse-width modulation (PWM). S74C164N shift register, input pins and output pins. S74C164N shift register to count from 0 to F (HEX) on a 7 segment common anode LED display. A 7-segment LED display is simply a group of light emitting diodes arranged in the pattern of an eight (plus decimal point) that will display numbers when the appropriate are turned on.

Expansion Ports :


This block consists of 28 pins divided into power pins, analog pins and digital pins.

RGB LED Interface :


This block consists of 9 common anode RGB LED slots. Every RGB LED has 3 R G B inputs.

LCD Interfacing :


This block consists of an LCD display, Potentiometer, BC547B transistor, input pins and output pins. It supports 4 data lines and 8 data line

4*4 Keyboard Interface :


This block consists matrix keyboard 4*4, 5 additional switches and input ports.

Relay, Buzzer and Servo Motor Interface :


This block consists of a relay pin, buzzer pin, servo pins, buzzer, relay and servo motor port. It has on board led which indicates relay switching.

Motor Interfaces (DC Motor & Stepper Motor) :


This block consists of L293DNE motor driver IC, dc motor port, stepper motor port, unipolar and bipolar motor ports.

IR Remote Interface :


This block consists of external IR sensor ports and internal IR sensor ports. It can be switched using the jumper section.
